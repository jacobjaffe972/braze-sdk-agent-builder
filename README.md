# AI Agents Demo

A production-ready AI agents framework demonstrating progressive enhancement of LLM capabilities through multiple agent architectures.

## Overview

This project showcases advanced AI agent design patterns, from basic query classification to sophisticated multi-agent research orchestration. Built with LangChain, LangGraph, and OpenAI APIs.

## Features

- **Progressive Agent Capabilities**: 9 different agent modes across 3 families
- **Multi-Agent Orchestration**: Research Manager → Specialized Researchers → Evaluator → Report Finalizer
- **RAG Implementation**: Document retrieval, web search, and hybrid corrective RAG
- **ReAct Pattern**: Autonomous reasoning and action with tool use
- **Production Observability**: Opik integration for tracing and monitoring

## Tech Stack

- **LLM**: OpenAI GPT-4 / GPT-4o-mini
- **Framework**: LangChain + LangGraph
- **Vector DB**: ChromaDB
- **Web Search**: Tavily
- **Observability**: Opik
- **UI**: Gradio
